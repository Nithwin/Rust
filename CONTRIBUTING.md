## Pull Request Process

1.  **Fork** the repository.
2.  **Clone** your fork locally.
3.  Create a **new branch** for your feature (`git checkout -b amazing-feature`).
4.  Commit your changes (`git commit -m 'Add some amazing feature'`).
5.  Push to the branch (`git push origin amazing-feature`).
6.  Open a **Pull Request**.

## Code of Conduct

Be kind. We are here to learn together. Constructive feedback is welcome; criticism without kindness is not.
